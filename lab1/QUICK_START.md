# Быстрый старт

## Запуск программы

### 1. Активация виртуального окружения

**macOS/Linux:**
```bash
source venv/bin/activate
```

**Windows:**
```bash
venv\Scripts\activate
```

### 2. Запуск программы

```bash
python lab1.py
```

или напрямую через venv:

```bash
./venv/bin/python lab1.py
```

## Результаты

После запуска программа выведет:
1. Подробный текстовый отчет в консоль
2. Создаст файл `statistical_analysis_results.png` с графиками

## Структура отчета

1. Исходные данные
2. Первоначальная оценка математического ожидания
3. 95% доверительный интервал
4. Отсеивание аномальных наблюдений
5. Уточненная оценка
6. Доверительный интервал для β = 0.89
7. Доверительная вероятность для εβ = 0.28
8. Проверка качества оценивания

## Ключевые результаты (Вариант 7)

- **Исходная оценка:** x̄ = 4.46 ± 3.50
- **Обнаружен выброс:** 12.5 (выходит за 95% прогнозный интервал [-3.55, 12.47])
- **Уточненная оценка:** x̄* = 3.73 ± 1.34 (при β = 0.89)
- **Влияние выброса:** смещение оценки на 16.4%
- **Вывод:** Качество оценивания НЕ удовлетворительное
  - Полученная погрешность (1.34) в 4.8 раза превышает требуемую (0.28)
  - Необходимо увеличить выборку до ~243 наблюдений
